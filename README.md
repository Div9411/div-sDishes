# div'sDishes - Recipe Search Platform

Built using HTML, CSS, and Vanilla JavaScript.

## Functionalities:

- Search recipes.
- Check details and ingredients.
- Add your favorite recipes to bookmarks.
- Remove unwanted bookmarks.
